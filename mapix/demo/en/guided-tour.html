This gallery is made using a simple HTML file (demo/en/gallery.html) and the gallery/script.js and style.css

  • Administration site

    Mapix CMS back office is based on file explorer and uses context menus, asynchronous multiple upload, drag and drop...

  • XML Document Editor

    Every document is editable online, using a basic XML editor : nodes can be folded/unfolded and sorted using drag and drop.

  • Site template (map files)

    The whole sitemap is based on XML documents : the "map files" contain a node for each page. Each page contain many templates and/or content and may call other HTTP services.

  • Text file editor

    Non-XML text files are automatically opened with a text editor : Edit_Area, an open source text editor.

  • Image editor

    Images are automatically opened with a basic image editor with rotate, crop and resize functions.
