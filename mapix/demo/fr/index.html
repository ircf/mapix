Mapix CMS : Une nouvelle génération de CMS

Mapix CMS est un gestionnaire de contenu simple, qui permet de gérer l'ensemble de votre site sans utiliser de base de données.

Tous les contenus de votre site sont stockés dans des documents XML et sont éditables en ligne (voir la démo du site d'administration).

Mapix CMS est un logiciel libre sous licence GNU GPL, simple d'utilisation et surtout extensible. Plusieurs extensions sont prévues dans les mois à venir : Forum, Blog, E-Commerce...

Vous souhaitez contribuer à développer Mapix CMS ou une extension, vous pouvez vous inscrire en tant que membre du projet Mapix CMS sur sourceforge.net.
