Cette galerie utilise un fichier HTML (demo/fr/visite-guidee.html) et le paquet gallery (gallery/script.js et style.css).

  • Site d'administration

    Le site d'administration de Mapix CMS est basé sur un explorateur de fichiers et utilise des menus contextuels, le téléchargement asynchrone multiple, le glisser-déposer...

  • Editeur de documents XML

    Tous les documents XML est éditable grâce à un éditeur XML en ligne : les éléments peuvent être dépliés/repliés ou déplacés en glisser-déposer.

  • Structure du site (fichiers map)

    L'ensemble du site est basé sur des fichiers XML : les fichiers "map" (plans) contiennent un élément pour chaque page. Chaque page contient un ou plusieurs gabarits et/ou contenus et peut faire appel à des services HTTP.

  • Editeur de fichiers textes

    Les fichiers textes sont ouverts automatiquement avec codepress, un éditeur texte libre.

  • Editeur d'images

    Les fichiers images sont ouverts avedc un éditeur simple qui contient les fonctions de base pour pivoter, recadrer et redimensionner une image.
