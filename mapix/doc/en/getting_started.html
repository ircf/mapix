Creating a new website

  • Go to http://localhost, and click on administration site
  • Login using "admin" and "demo". You can now browse your document root using the folder editor.
  • To begin, browse the mapix folder, copy the demo folder (right-click > Copy), and paste it to a new "yoursite" folder inside the root folder (you should NOT paste the "yoursite" folder in the mapix folder).
  • Get back to the root folder, open the controller.xsl file, replace "mapix/demo" by "yoursite", and save file.
  • You can now create/modify/remove HTML pages in the "yoursite" folder, while your site still uses mapix theme, be sure to keep a menu.html and a 404.html page.

Creating a custom theme

To create your custom theme, go to mapix folder again, and copy the theme dir, and paste it into the "yoursite" folder. Here we assume this theme will only be used by "yoursite". If you plan to reuse your theme for many sites (make a "standard" theme instead of a "custom" theme), then you can paste it in the mapix folder.
